## Customization

### Update Personal Information

Edit `index.html` to update:
- Name and title in the hero section (line 34)
- About section content (lines 61-64)
- Project details (lines 75-143)
- Skills list (lines 157-190)
- Contact information (lines 201-235)

### Change Colors

Edit `styles.css` CSS variables (lines 7-14):
```css
:root {
    --primary-color: #6366f1;
    --secondary-color: #8b5cf6;
    /* Add your custom colors */
}
```
